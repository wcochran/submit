CS 330 Project Submission Form

The following form is intended to allow for easy submission of programming projects for the CS 330 Numerical Computing course. All the necessary files need to be archived into a single compressed archive file which is then submitted via this form. The following file formats are acceptable:

  • GZIP'ed TAR file (.tar.gz),
  • ZIP (.zip),
  • JAR (.tar).

From a UNIX shell you can do one of the following:

  • To create a gzipped tarball named f.tar.gz: 
    tar czvf f.tar.gz fred.c wilma.h README
  • To create a .zip file named f.zip: 
    zip f.zip fred.c wilma.h README
  • To create a Java Archive (JAR) file named f.jar: 
    jar f.jar fred.java wilma.java README

(8 digits, no leading zero)
