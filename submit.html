CS 330 Project Submission Form

The following form is intended to allow for easy submission of programming projects for the CS 330 Numerical Computing course. All the necessary files need to be archived into a single compressed archive file which is then submitted via this form. A compressed Unix tarball, using either compress or gzip, or a PKZIP file is acceptable. To create a compressed tarball containing the files fred.c and wilma.h do the following: 
tar cvf - fred.c wilma.h | compress > f.tar.Z
To create a gzipped tarball do the following: 
tar czvf f.tar.gz fred.c wilma.h
To create a .zip file under DOS/Win32, get a copy of winzip, PKZIP, or find a shareware version somewhere. In Linux or OS X, you can do the following: 
zip f.zip fred.c wilma.h
